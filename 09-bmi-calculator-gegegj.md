# BMI 指数计算｜鸽鸽工具网

## 功能介绍

BMI（Body Mass Index，体质指数）= 体重(kg) ÷ 身高(m)²。页面根据计算结果给出中文区间标签：

- **偏瘦**：BMI < 18.5  
- **正常**：18.5 ≤ BMI < 24  
- **超重**：24 ≤ BMI < 28  
- **肥胖**：BMI ≥ 28  

（区间为中国成人常用参考分段之一，**非诊疗标准**的全部内涵。）

## 支持的参数

| 参数 | 说明 |
|------|------|
| 身高 | 厘米（cm） |
| 体重 | 千克（kg），支持一位小数 |

## 常见问题

**BMI 正常就一定健康吗？**  
不一定。不反映肌肉量、内脏脂肪、代谢疾病风险等。

**健身人群 BMI 偏高？**  
肌肉密度大可能导致体重偏高，BMI 偏高未必代表体脂过高。

**儿童、孕妇能用吗？**  
不建议套用成人分段；应使用儿科学会生长曲线或产科营养指导。
