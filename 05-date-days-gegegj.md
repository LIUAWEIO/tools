## 功能介绍

选择两个**公历日期**（开始日期、结束日期），计算二者之间相差的**日历天数**（取绝对间隔，与先后顺序无关的展示逻辑以页面为准）。适合快速回答「相差几天」类问题。

在项目管理、行政人事与合同管理中，常见说法包括：**「两个日期相差几天」「日期间隔计算器」「工期多少天」「合同天数怎么算」「请假天数」** 等。本站给出**日历日**间隔；若您需要**工作日**（剔除周末与法定节假日），请结合企业日历或人事制度另行处理。

## 支持的参数

| 参数 | 说明 |
|------|------|
| 开始日期 | `type="date"` 选择器 |
| 结束日期 | 同上 |

内部按 UTC 日边界对齐日期部分，避免夏令时边界对「日」的干扰（实现为将本地日期串解析为当日中午再取 UTC 日差）。

## 日历日 vs 工作日 vs「含首尾」

| 口径 | 说明 |
|------|------|
| 日历日 | 本站默认输出的间隔，连续自然日计数。 |
| 工作日 | 需剔除周末、法定节假日与调休，本站不内置各地假日库。 |
| 含首尾日 | 合同可能约定「自起算日起算满 N 日含当日」，需在间隔结果上按规则 ±1 或按文本解释处理。 |

## 使用场景

- 项目排期、活动筹备：粗算持续天数。
- 合同、租赁、保修：核对起止日之间的日历跨度。

## 常见问题

**结果是否包含开始日和结束日两天？**  
默认多为两日之间的**间隔天数**，不一定等于「头尾都算」的自然日数。合同写「自 X 至 Y 共 N 日」时请以文本解释为准。

**能算工作日吗？**  
当前为**日历日**，未排除周末与法定节假日。工作日需结合企业日历另行计算。

**跨年、闰年可以吗？**  
可以，只要日期合法。

**和 Excel DATEDIF 不一致？**  
不同软件对「是否含首日」与日期序列基准可能不同；请以贵司规定或合同定义为准。
