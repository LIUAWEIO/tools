# 日期天数计算器｜鸽鸽工具网

**立即使用：** [https://gegegj.com/calc/date](https://gegegj.com/calc/date)

## 功能介绍

选择两个**公历日期**（开始日期、结束日期），计算二者之间相差的**日历天数**（取绝对间隔，与先后顺序无关的展示逻辑以页面为准）。适合快速回答「相差几天」类问题。

## 支持的参数

| 参数 | 说明 |
|------|------|
| 开始日期 | `type="date"` 选择器 |
| 结束日期 | 同上 |

内部按 UTC 日边界对齐日期部分，避免夏令时边界对「日」的干扰（实现为将本地日期串解析为当日中午再取 UTC 日差）。

## 使用场景

- 项目排期、活动筹备：粗算持续天数。
- 合同、租赁、保修：核对起止日之间的日历跨度。
- 请假统计：先算日历差，再结合人事制度是否 +1 或扣减首尾。

## 如何使用

1. 打开 [日期天数计算器](https://gegegj.com/calc/date)。
2. 分别点选开始日期与结束日期。
3. 查看「相差 X 天」类结果；若业务要求「含头含尾」，请在理解默认口径后自行 ±1。

## 常见问题

**结果是否包含开始日和结束日两天？**  
默认多为两日之间的**间隔天数**，不一定等于「头尾都算」的自然日数。合同写「自 X 至 Y 共 N 日」时请以文本解释为准。

**能算工作日吗？**  
当前为**日历日**，未排除周末与法定节假日。工作日需结合企业日历另行计算。

**跨年、闰年可以吗？**  
可以，只要日期合法。

## 数据来源与计算依据

不使用外部天文或节假日 API；纯本地日期运算。
