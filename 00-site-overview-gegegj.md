## 支持的「参数」与能力边界（整站级说明）

- **输入形态**：各页提供数字、日期、时间、文本或 JSON 等对应输入项；具体字段以各工具页为准。
- **房贷**：贷款金额（万元）、年利率（%）、贷款年限（年）、等额本息/等额本金、商贷/公积金参考利率快捷切换。商业性个人住房贷款利率定价与 **LPR（贷款市场报价利率）** 及合同约定加点相关，**请以贷款合同与银行系统为准**；公开信息可查阅 [中国人民银行](http://www.pbc.gov.cn/) 及全国银行间同业拆借中心公布的 LPR 形成机制与最新报价。
- **个税**：税前月工资、五险一金扣除、专项附加扣除（月）；年终奖单独计税模式。税制与专项附加政策可能随年度调整，**请以国家税务总局最新政策及单位财务口径为准**。
- **人民币大写**：金额（元，支持两位小数）。
- **汇率**：源/目标币种、金额；实时汇率（需网络）或手动录入「1 人民币 = X 外币」基准表。银行现钞/现汇、买卖价差与手续费会导致与页面参考价不同。
- **日期/年龄**：公历日期选择；年龄支持「计算基准日」。
- **时间戳**：10 位秒 / 13 位毫秒，或本地 `YYYY-MM-DDTHH:mm` 与 Unix 互转。
- **时区**：本地日期时间或 Unix 为「同一时刻」，对照多个 IANA 时区。
- **BMI**：身高（厘米）、体重（千克）。
- **预产期计算器 / 孕周计算器 / 孕期计算器**：末次月经第 1 天、月经周期（20～60 天）；本地按预产期计算公式（Naegele）推算预产期、孕周计算与怀孕多少周，支持怀孕周期备忘，**不构成医疗诊断**。
- **单位换算器**：长度单位换算（米/千米/厘米/英尺/英寸）、重量单位换算（千克/克/磅/市斤）、温度单位换算（华氏度和摄氏度）；搜索词「体积单位换算」「面积单位换算」常见，本站当前仅支持上述三类。
- **乘法口诀表 / 九九表**：乘法表算式阶梯、乘法口诀对照；可下载九九乘法表图片/乘法口诀表图片便于打印；附大九九乘法表与小九九，浏览器本地展示。
- **二维码生成器 / 条形码生成器**：二维码在线生成、网址转二维码；商品编码、订单号等生成 CODE128、EAN、UPC 等条码，支持批量导出。
- **压缩图片（图片压缩工具）**：JPG、PNG、WebP 批量上传，本地压缩后单张下载或打包 ZIP。
- **JSON 格式化工具**：json 在线解析，美化、压缩、json validator 校验与 json viewer 折叠查看。
- **字数统计工具**：字数统计在线/在线字数统计，查字数与字符统计；粘贴全文后本地统计，不上传正文。

## 数据来源与计算说明

- **汇率（实时模式）**：由服务端调用 [Frankfurter API](https://api.frankfurter.app/)（`from=CNY`）拉取，失败时尝试 [open.er-api.com](https://open.er-api.com/) 备用；以人民币为基准做交叉换算，响应缓存约 5 分钟。
- **个税**：采用综合所得七级超额累进税率表的**简化估算模型**（月应纳税所得额×12 套年表再÷12），**未完整模拟累计预扣逐月变化**；年终奖按单独计税常见算法演示。
- **房贷**：标准等额本息/等额本金数学模型，不含银行手续费、宽限期等合同细节。
- **时区**：各时区显示时间由浏览器 `Intl` 与系统时区数据计算，夏令时以系统数据为准。
- **Unix 时间戳**：基于 JavaScript `Date`，与 Epoch（1970-01-01 UTC）定义一致。
- **预产期计算器**：末次月经 + 预产期计算公式（Naegele）及周期简易顺延，支持孕周计算与怀孕多少周展示，未接入医院信息系统。
- **字数统计工具**：按页面公开规则对 Unicode 文本分类计数，与 Word 等软件口径可能不同。
