# 只要工作日天数？先用日历日估算再扣假

**在网站上阅读：** [https://gegegj.com/blog/date-workdays-pointer](https://gegegj.com/blog/date-workdays-pointer)

**相关工具：** [打开工具页](https://gegegj.com/calc/date)

## 推荐工作流

先用日历日得到总跨度，再列出需要扣除的周末与节假日天数，得到近似工作日。

项目里程碑若要以「自然日」对外承诺，可同时标注 [Unix 时间戳](https://gegegj.com/calc/timestamp) 便于研发对齐日志。

## 与排期发布

全球发布窗口可再用 [时区换算](https://gegegj.com/calc/timezone) 写进文档，减少执行误差。
