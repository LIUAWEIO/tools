# Wi-Fi 与电子名片二维码：内容越短越容易扫

**在网站上阅读：** [https://gegegj.com/blog/qrcode-wifi-and-vcard](https://gegegj.com/blog/qrcode-wifi-and-vcard)

**相关工具：** [打开工具页](https://gegegj.com/tool/qrcode)

## 内容建议

Wi-Fi 字符串与 vCard 字段尽量精简；长文本考虑拆页或短链。详见 [二维码生成器教程](https://gegegj.com/blog/qrcode-generator-guide)。

## 内容很长时

若扫出来的内容是很长的设置类文字，可先用 [JSON 格式化工具](https://gegegj.com/tool/json) 做 json validator 检查，再生成二维码。
