# 收款码换成彩色或加了 logo，顾客还能扫吗？

**在网站上阅读：** [https://gegegj.com/blog/qrcode-collection-code-style-still-scan](https://gegegj.com/blog/qrcode-collection-code-style-still-scan)

**相关工具：** [打开工具页](https://gegegj.com/tool/qrcode)

## 扫码靠对比度，不靠花纹

二维码本质是黑白模块；浅色底、低对比、过度渐变会让摄像头难对焦。中心放 logo 可以，但要留**静默区**（四周空白），别贴满图案。

用 [二维码生成器](https://gegegj.com/tool/qrcode) 生成后，先用微信/支付宝自己扫通，再印刷；打印尺寸过小、反光、皱褶都会失败，见 [扫不上](https://gegegj.com/blog/qrcode-scan-fail-blur-size-light)。

中心 logo 与静默区细节见 [logo 与静默区](https://gegegj.com/blog/qrcode-logo-center-quiet-zone)。

## 和静态海报、WiFi 码的区别

收款码多是支付平台动态规则，别用普通 URL 码冒充；活动链接过期见 [链过期](https://gegegj.com/blog/qrcode-scans-but-link-expired)。

静态海报长期不换链接见 静态海报；WiFi 码见 WiFi vCard。

## 打印与张贴注意

收款码打印太小、反光、皱褶都会扫不上；留白不够时部分 App 容错也救不了，先手机屏扫通再印。

店铺同时贴微信与支付宝，是两码还是合一码，以支付平台说明为准，别用普通网址二维码冒充收款码。

活动海报上二维码过期后，见 链已过期；中心 logo 见 静默区。

WiFi 码、名片码见 WiFi 与 vCard，字段格式错误会显示已连接却无法上网。

## 店铺经营里的替换节奏

换收款码前先贴新码旁测一周，旧码再撤，避免顾客习惯扫旧码失效。

亚克力立牌反光严重，换磨砂或调整角度；夜市灯光不足时加小射灯照码面。

合并微信支付宝为一张「聚合码」时，确认手续费与到账账户，别只图海报好看。

活动临时码过期后，海报上旧码未撕，顾客扫进失效页，见 链接过期。

外卖平台抽成与线下收款码是两套账，别用美化收款码海报代替平台订单二维码。

## 使用提醒

本文只讨论标题里的一个具体场景，二维码生成器 在浏览器本地计算或处理，适合自查与沟通；签约、报税、诊疗、签证请以主管机构或合同为准。

涉及隐私的数据请先脱敏；公共电脑、录屏环境勿粘贴真实工资、病历、Token 或合同金额。

## 小结

美化收款码：高对比、留静默区、logo 别遮关键角、先屏扫后印刷。

支付规则以平台为准，本文谈通用扫码物理因素。
