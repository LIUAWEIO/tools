# 孩子乘法口诀总错「7」这一行，有没有系统的练法？

**在网站上阅读：** [https://gegegj.com/blog/multiplication-wrong-row-pattern-fix](https://gegegj.com/blog/multiplication-wrong-row-pattern-fix)

**相关工具：** [打开工具页](https://gegegj.com/tool/multiplication-table)

## 先找「模式」不是粗心

总错 7×8、7×9，却不错 8×7，可能是某一行的顺序记忆断了；总把 6×7 答成 48，可能是和 6×8 干扰。记录一周错题，标出式子。

打开 [乘法口诀表](https://gegegj.com/tool/multiplication-table)，只显示或口述「7 的口诀」一行，配合书写 7×1 到 7×9，再混 6、8 两行做对比，避免一次 81 格全背。

会背但除法慢，见 [会乘不会除](https://gegegj.com/blog/multiplication-fluent-but-division-slow)；7×8 与 6×8 混见 [7×8](https://gegegj.com/blog/multiplication-7x8-56-confused-with-6x8)。

## 家长怎么陪练不添压

用口算卡片或手指游戏，答对具体式子就夸，别说「你怎么7都不会」；错因描述清楚，孩子才知道改哪一格。

跳行背漏见 [跳行查漏](https://gegegj.com/blog/multiplication-table-skip-row-check)；大九九小九九区别见 大九小九。

## 一周纠正计划（家长向）

第1–2天只练错的那一行（如 7 的口诀）， orally + 书写；第3天混入相邻行（6 和 8）防干扰。

用 乘法口诀表 按行高亮，别一次竖背 81 格；跳行查漏见 跳行背错。

错因若是「6×7 和 7×6 搞反」，说明交换律已懂，加强较大数在前的心算即可。

每天十分钟比周末恶补一小时更有效；别在错题上贴标签，描述具体式子。

## 和老师作业要求对齐

老师要求「一分钟口算二十题」时，错题集中在同一行，就专练那一行一分钟；达标后再混全表。

竖式乘法进位错与口诀表是不同阶段问题；口诀表练的是即时回忆，竖式练的是位值，别混为一谈。

用实物分组（如七颗糖每组八个）帮助理解 7×8，再背口诀，比死记更易巩固。

错题本只抄式子不抄「我笨」，孩子更愿意复习；家长签字关注具体哪两式。

假期返校前一周每天十分钟复习，比开学前一天突击更有效；见 跳行查漏。
