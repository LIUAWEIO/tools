## 功能介绍

本页是**预产期计算器**，同时覆盖**孕周计算器**、**孕期计算器**与**怀孕计算器**常用需求：根据**末次月经第 1 天**与**月经周期（20～60 天）**，在浏览器本地推算：

- **预产期**：常用预产期计算公式（Naegele：末次月经起加 1 年、减 3 个月、再加 7 天），并按周期相对 28 天做简易顺延。  
- **孕周计算 / 怀孕多少周**：自末次月经起算的「第几周+几天」。  
- **怀孕周期备忘**：显示距预产期剩余天数或已超过天数。

全部计算在**本机浏览器**完成，不向本站上传末次月经等隐私字段。结果**仅供生活备忘**，不能替代产检与医生诊断。

## 支持的参数

| 参数 | 说明 |
|------|------|
| 末次月经第 1 天 | 公历日期选择。 |
| 月经周期 | 20～60 天可选，默认 28 天；用于相对标准周期做简易顺延。 |

## 预产期计算公式（Naegele）

| 步骤 | 说明 |
|------|------|
| 基准 | 末次月经第 1 天 |
| 规则 | +1 年、−3 个月、+7 天 |
| 周期顺延 | 相对 28 天周期增减天数（经验算法） |

## 为什么和医院 B 超单可能不一致？

| 原因 | 说明 |
|------|------|
| 末次月经记忆偏差 | 日期差几天会带动预产期与孕周整体偏移。 |
| 周期不规律 | 公式为经验推算，不如早孕期超声校正准确。 |
| 辅助生殖受孕 | 受孕日与自然周期不同，应以生殖中心与产科医嘱为准。 |

## 常见问题

**怀孕多少周怎么查？**  
选择末次月经后计算，页面显示「第几周+几天」；与产检 B 超有差异时以医院记录为准。

**超过预产期还没生怎么办？**  
预产期前后 1～2 周分娩都较常见；是否干预由医院评估，请勿自行判断。

**能替代产检吗？**  
不能。本工具不提供医学建议，超声与产科随访才是金标准。

**会上传我的末次月经吗？**  
不会。计算在本地完成，本站不接收表单中的日期作为业务数据存储（以当前版本说明为准）。
