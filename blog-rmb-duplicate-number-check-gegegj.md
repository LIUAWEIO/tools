# 合同里小写与大写不一致怎么办？

**在网站上阅读：** [https://gegegj.com/blog/rmb-duplicate-number-check](https://gegegj.com/blog/rmb-duplicate-number-check)

**相关工具：** [打开工具页](https://gegegj.com/calc/rmb)

## 推荐核对顺序

先锁定小写金额与币种，再生成大写；最后从高位到低位逐字对照「亿万千百十元角分」。

涉及期限与付款节点时，可用 [日期天数](https://gegegj.com/calc/date) 辅助算周期天数，但条款解释以法务为准。

## 与房贷、个税交叉

贷款合同常见大小写双写，可结合 [房贷计算器](https://gegegj.com/calc/mortgage) 核对月供相关数字是否与合同摘要一致。
