# 从超大日志里截取一段 JSON：先截断再格式化

**在网站上阅读：** [https://gegegj.com/blog/json-log-snippet-workflow](https://gegegj.com/blog/json-log-snippet-workflow)

**相关工具：** [打开工具页](https://gegegj.com/tool/json)

## 推荐做法

先定位错误行附近几百行，再粘贴进工具。基础能力见 [JSON 格式化工具指南](https://gegegj.com/blog/json-formatter-guide)。

## 与时间戳

日志行里若带 Unix 时间，可用 [时间戳换算](https://gegegj.com/calc/timestamp) 对齐人类可读时间。
