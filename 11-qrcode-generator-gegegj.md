# 二维码生成｜鸽鸽工具网

**立即使用：** [https://gegegj.com/tool/qrcode](https://gegegj.com/tool/qrcode)

## 功能介绍

基于 `qrcode` 库在**用户浏览器本地**生成二维码画布，可导出 **PNG**（`toDataURL`）。支持在中间叠加：

- **站点品牌图标**（默认 favicon），或  
- **用户本地上传的图片**  

并可调节图标占画布比例（约 8%～34%）、圆角比例、白边（padding）比例；过大图标会触发页面上的扫码风险提示。

## 支持的参数

| 参数 | 说明 |
|------|------|
| 二维码内容 | 任意文本或 URL；**内容越长，码越密** |
| 中间图标开关 | 关闭则为标准黑白码 |
| 图标模式 | 品牌图标 / 本地上传 |
| 图标大小、圆角、白边 | 百分比滑块，影响可识别性 |
| 输出 | 画布宽度 560px、margin 1、深色 `#0f172a` |

页面说明：**不会在服务器端存储您输入的载荷**（绘制在本地完成）。

## 常见问题

**Logo 多大合适？**  
经验上约占二维码宽度 **18%～24%**，并保留足够**静区（白边）**；页面在占比过大时会提示可能影响识别。

**为什么扫不出来？**  
常见原因：内容过长、打印分辨率不足、对比度低、Logo 遮挡纠错区。

**能商用吗？**  
请自行确认**目标链接、商标、字体与素材版权**及当地广告法规；本站仅提供生成工具。
