# 在线房贷计算器（鸽鸽工具网）

**立即使用：** [https://gegegj.com/calc/mortgage](https://gegegj.com/calc/mortgage)

## 功能介绍

在浏览器中完成**商业贷款 / 公积金参考**场景下的月供试算，支持**等额本息**与**等额本金**两种还款方式，输出月供、还款计划相关汇总（以页面实际展示为准），便于购房前评估现金流与对比不同贷款方案。

商业性个人住房贷款利率与 **LPR（贷款市场报价利率）** 及合同约定的加减点密切相关。LPR 由中国人民银行授权全国银行间同业拆借中心于**每月 20 日（遇节假日顺延）**公布；**5 年期以上 LPR** 常作为房贷定价的重要参考之一。**具体执行利率、折扣、重定价周期与还款日**以您与银行签署的借款合同及银行系统为准，可查阅 [中国人民银行](http://www.pbc.gov.cn/) 发布的最新政策与报价信息。

## 支持的参数

| 参数 | 说明 |
|------|------|
| 贷款金额 | 以「万元」为单位输入 |
| 年利率 | 百分比，如 3.1 表示 3.1%/年 |
| 贷款年限 | 年，页面校验为正数（常见 1～30 年等） |
| 还款方式 | 等额本息（EPI）/ 等额本金（EP） |
| 贷款类型快捷 | 「商贷参考」默认示例利率、「公积金参考」另一组示例利率；**均可手动改利率** |

商贷/公积金按钮仅便于快速填入**示例**利率，实际利率请以当地政策与银行报价为准。

## 使用场景

- 估算 100 万、200 万等不同贷款额度在 20 年、30 年下的月供。
- 对比等额本息与等额本金的总利息与前期月供压力。
- LPR 或利率上浮/下调后的敏感性试算。
- 与家庭收入、公积金缴存结合，粗算「月供占收入比」；税后收入可配合 [个税计算器](https://gegegj.com/calc/tax) 估算。

## 示例思路（非合同结果）

假设贷款 **100 万元**、年利率 **3.1%**、期限 **30 年**：可在页面分别切换等额本息 / 等额本金，观察**首月月供、末月月供（等额本金递减）**、**总利息**差异。再尝试把年利率改为 **3.5%** 或 **2.85%**，即可得到一版「压力测试」结果，用于判断利率波动对现金流的影响。示例数字**不代表**您能获得的真实利率。

## 如何使用

1. 打开 [房贷计算器页面](https://gegegj.com/calc/mortgage)。
2. 输入贷款金额（万元）、年利率（%）、贷款年限（年）。
3. 选择商贷或公积金参考（或保留自定义利率），再选择等额本息或等额本金。
4. 查看月供、总还款与总利息等结果；可按需使用复制功能。

组合贷可将商贷与公积金部分**分别试算**后相加，作为近似参考。

## 常见问题

**等额本息和等额本金哪个更省利息？**  
在金额、利率、期限相同的前提下，等额本金通常总利息更低，但前期月供更高；等额本息月供更稳定。选择取决于现金流偏好与收入结构。

**计算结果能直接作为银行合同金额吗？**  
不能。实际月供受审批利率、放款日、还款日、优惠及合同条款影响，请以借款合同与银行系统为准。

**支持组合贷一键计算吗？**  
当前以单一路径试算为主；组合贷建议分笔计算后自行加总。

**提前还款怎么估？**  
本站试算基于标准还款计划，**未内置**各家银行提前还款违约金、缩短期限不变月供等细则；请向贷款银行确认规则后再决策。

**首套房、二套房利率不同怎么办？**  
请按银行告知的实际年利率直接填入「年利率」栏分别试算，不要依赖页面示例默认值。
