# 在线房贷计算器（鸽鸽工具网）

**立即使用：** [https://gegegj.com/calc/mortgage](https://gegegj.com/calc/mortgage)

## 功能介绍

在浏览器中完成**商业贷款 / 公积金参考**场景下的月供试算，支持**等额本息**与**等额本金**两种还款方式，输出月供、还款计划相关汇总（以页面实际展示为准），便于购房前评估现金流与对比不同贷款方案。

## 支持的参数

| 参数 | 说明 |
|------|------|
| 贷款金额 | 以「万元」为单位输入 |
| 年利率 | 百分比，如 3.1 表示 3.1%/年 |
| 贷款年限 | 年，页面校验为正数（常见 1～30 年等） |
| 还款方式 | 等额本息（EPI）/ 等额本金（EP） |
| 贷款类型快捷 | 「商贷参考」默认示例利率、「公积金参考」另一组示例利率；**均可手动改利率** |

商贷/公积金按钮仅便于快速填入**示例**利率，实际利率请以当地政策与银行报价为准。

## 使用场景

- 估算 100 万、200 万等不同贷款额度在 20 年、30 年下的月供。
- 对比等额本息与等额本金的总利息与前期月供压力。
- LPR 或利率上浮/下调后的敏感性试算。

1. 打开 [房贷计算器页面](https://gegegj.com/calc/mortgage)。

组合贷可将商贷与公积金部分**分别试算**后相加，作为近似参考。

## 常见问题

**等额本息和等额本金哪个更省利息？**  
在金额、利率、期限相同的前提下，等额本金通常总利息更低，但前期月供更高；等额本息月供更稳定。

**计算结果能直接作为银行合同金额吗？**  
不能。实际月供受审批利率、放款日、还款日、优惠及合同条款影响，请以借款合同与银行系统为准。

**支持组合贷一键计算吗？**  
当前以单一路径试算为主；组合贷建议分笔计算后自行加总。

## 数据来源与计算依据

计算在**用户浏览器本地**完成，使用标准金融公式（等额本息、等额本金），**不包含**保险费、担保费、提前还款罚息等银行附加规则。
